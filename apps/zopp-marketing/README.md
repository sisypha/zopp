# zopp Marketing Site

Marketing landing page for zopp, deployed at [zopp.dev](https://zopp.dev).

## Tech Stack

- [Astro](https://astro.build/) - Static site generator
- [Tailwind CSS](https://tailwindcss.com/) - Styling
- Same design tokens as the main web app

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

The site is deployed as a static site. The build output is in `dist/`.
